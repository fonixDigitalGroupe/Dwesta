# DWESTA Solutions

Site vitrine statique pour DWESTA Solutions - Solutions digitales innovantes.

## 🚀 Description

DWESTA Solutions est une société de conseil en transformation numérique avec plus de 10 ans d'expertise. Ce site vitrine présente nos services et notre expertise dans le domaine des solutions digitales.

## ✨ Fonctionnalités

- **Site vitrine responsive** avec design moderne
- **Navigation intuitive** avec menu mobile
- **Sections principales** :
  - Accueil avec bannière hero
  - À propos avec présentation de l'équipe
  - Services avec nos offres
  - Contact avec formulaire
- **Design professionnel** avec couleurs corporate
- **Optimisé mobile** et desktop

## 🛠️ Technologies

- **Laravel 11** - Framework PHP
- **Tailwind CSS** - Framework CSS
- **Blade** - Moteur de template

## 🚀 Installation

1. **Cloner le repository**
   ```bash
   git clone https://github.com/fonixDigitalGroupe/Dwesta.git
   cd Dwesta
   ```

2. **Installer les dépendances**
   ```bash
   composer install
   npm install
   ```

3. **Configurer l'environnement**
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

4. **Compiler les assets**
   ```bash
   npm run build
   ```

5. **Démarrer le serveur**
   ```bash
   php artisan serve
   ```
